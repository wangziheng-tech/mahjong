# Full code and model

This directory contains the complete V7+call150 code/model package used for the
final submission.

Important files:

- `models/mahjong_v7_calibrated_best.pkl`: Botzone storage model.
- `models/mahjong_v7_calibrated_best.pt`: same weights saved with `.pt` suffix.
- `botzone/mahjong_v7_call150_friendstyle_storage_bot.zip`: no-model Botzone package.
- `botzone_final/`: final Botzone source.
- `training_code/`: training, diagnostics, self-play, and support scripts.
- `eval_results/`: key self-play/evaluation outputs.
